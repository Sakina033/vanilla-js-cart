# Корзина покупок на чистом JavaScript

Это простой проект корзины покупок, реализованный на чистом JavaScript, без использования фреймворков и библиотек.

## О проекте

Проект был создан в целях практики и углубления знаний чистого JavaScript. Основное внимание уделялось логике работы корзины: добавлению, удалению, изменению количества товаров, а также подсчету общей стоимости. Дизайн не является приоритетом, так как я не веб-дизайнер и фокусировалась исключительно на функциональной части JavaScript.

### Основные функции:
* Добавление товаров в корзину
* Удаление товаров из корзины
* Изменение количества товаров
* Подсчет общей стоимости

## Технологии

* HTML
* CSS
* JavaScript (Vanilla JS)

# Shopping Cart in Pure JavaScript

This is a simple shopping cart project implemented using pure JavaScript, without any frameworks or libraries.

## About the Project

This project was created for practice and to deepen my understanding of vanilla JavaScript. The main focus was on the shopping cart's logic: adding, removing, changing product quantities, and calculating the total cost. Design was not a priority as I am not a web designer and focused exclusively on the functional JavaScript part.

### Key Features:
* Adding items to the cart
* Removing items from the cart
* Changing item quantities
* Calculating the total cost

## Technologies

* HTML
* CSS
* JavaScript (Vanilla JS)
